Esri Global Navigation

"Web" mode is intended for use in general web sites (marketing, event, etc). It is very similar to app mode, but has a couple differences:

  • Main menu items float left
  • Slightly taller
  • No branded stripe across the top

The main menu items fold into the left hamburger menu on smaller screens.

Note: when using the tiles property inside a menu object, you are limited to showing a maximum of four tiles. The tiles should be items you wish to have more prominence in the visual hierarchy.

If you only have an icon/logo instead of a logotype, your website title can be added using the brandText property in the brand area of the header options object.

To report bugs, please open an issue.

To learn more about this demo, you can view the full JavaScript source on GitHub.
