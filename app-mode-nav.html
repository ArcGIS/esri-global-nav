App Mode Navigation

App mode is intended for use in SaaS applications. This mode is great for single-page applications and products. It is very similar to web mode, but has a couple differences:

  • Main menu items float right
  • Slightly shorter and more compact
  • Branded stripe across the top

The branded stripe and logo area should reflect your product's branding. If you don't have a logo and brand color for your particular SaaS application, the Creative Lab can help you and your team get one.

The main menu items fold into the left hamburger menu on smaller screens.

If you only have an icon/logo instead of a logotype, your product name can be added using the brandText property in the brand area of the header options object.

To report bugs, please open an issue.

To learn more about this demo, you can view the full JavaScript source on GitHub.
